CSS3 Filte

语法 

elm {
    filter: none | <filter-function > [ <filter-function> ]* 
    }

其默认值是none，他不具备继承性，其中filter-function一个具有以下值可选： grayscale灰度 sepia saturate饱和度 hue-rotate色相旋转 invert反色 opacity透明度 brightness亮度 contrast对比度 blur模糊 drop-shadow阴影
